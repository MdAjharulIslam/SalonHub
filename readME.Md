# SalonHub - Salon & Spa Booking Application

 [SalonHub link ] https://salonhub-lac.vercel.app   

**SalonHub** is a modern, responsive, and feature-rich salon and spa booking platform built using the **MERN stack** (MongoDB, Express, React, Node.js). Users can browse services, book appointments, and manage their bookings, while admins can manage services, view all bookings, and control availability.

## Project Overview

SalonHub simplifies salon and spa bookings. Users can:  

- View services such as Haircut, Hair Color, Hair Treatment, Facial, Massage, Makeup, Bridal, Hair Spa, etc.  
- Book services by selecting date, time, and providing contact info.  
- Check booking statuses (Pending, Confirmed, Cancelled).  

Admins can:  

- Add, edit, or delete services.  
- Toggle service availability.  
- View all bookings and update statuses.  
- Manage salon branches with map locations.

---

## Features

### User Features
- Browse services with descriptions and images.  
- Book appointments online.  
- View and manage personal bookings.  
- Get notifications on booking status updates.  
- Interactive map showing branch locations.  

### Admin Features
- Admin login and JWT authentication.  
- Add, update, or delete services.  
- Toggle service availability (active/inactive).  
- View all bookings in a dashboard.  
- Update booking statuses: Pending, Confirmed, Cancelled.  
- Branch management with map integration.

## Tech Stack

**Frontend:**  
- React.js with TypeScript  
- Tailwind CSS  
- Swiper.js (for banners/carousels)  
- React Leaflet (for map integration)  
- React Hot Toast (for notifications)  

**Backend:**  
- Node.js with Express.js  
- MongoDB with Mongoose  
- JWT for authentication  

**Other Tools:**  
- Axios for HTTP requests  
- Multer for image uploads  
- Vercel for frontend deployment  
- Render/Heroku for backend deployment
